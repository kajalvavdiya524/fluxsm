FAQ

  • I can't lose my current customer and vehicle history. Can data be transferred?
    Yes! If you can export your existing customer details, vehicle history, inventory, and vendors, we can get it loaded into Flux for a seamless transition.
  • For multiple locations, do I need multiple licenses?
    Each location would require a separate license, but additional licenses are at a discount. Even though separate licenses are needed, Flux will not run independently. Each location will have access to inventory from the other locations.
  • What is required to print work orders, estimates, and invoices?
    Any printer, inkjet or laser, would be compatible. Printing allows store-specific information, including a logo, to be added so all customer copies look professional.
  • What are the system requirements?
    For the cloud edition of Flux, the only requirement is an internet connection and a web browser. If you can get to this website, you will be able to use Flux. For the on-premise edition, you need a network router with an open ethernet port. Everything else is self-contained.
  • Does Flux include a mobile app?
    Flux does not have a separate mobile app, but Flux is mobile designed. Any device size from a small phone to a tablet to a wide screen desktop monitor will be able to use Flux without any restrictions or disabled functionality.
  • Is training required before being able to use Flux?
    Flux is designed to be a simple fill-in-the-blanks system. Training isn't required to be able to get up and running on the system.
